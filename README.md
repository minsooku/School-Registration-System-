# School-Registration-System-

# Required 
* Import Collection, HashMap, HashSet, Map, Scanner, Set;
* Extend Thread class

# How to use
* Add courses to the system as an array consisting course title, number and maximum number of students
* Add each student to a course that the student wants to take

# Description
* Utilized Maps and Sets to efficiently store the data for registration system (Courses, classes, students)
* Utilized static inner class that extends thread to run threads concurrently.
